GNU - распределенная система контроля версий
Это система для отслеживания и ведения истории изменения файлов, в вашем проекте.

Репозитория - хранилище вашего кода и истории его изменений.

GitHub - Веб сайт для размещения git-репозиториев в совместной разработке проектов.

Команды Git:

1. git init - она создаёт новый локальный Git-репозиторий у вас на ПК, в той директории в которой была прописана команда.(Команда)
2. git add - команда для добавления файлов в систему отслеживания гита, после этой команды изменения начинают отслеживаться гитом.
3. git commit -m '<comment>' - как только достигаем определенного момента в разработке, то тогда мы сохраняем и комментируем изменения в локальной репозитрий.
4. git remote add - это команда для того что бы связать наш локальный репазиторий с удаленным репозиторием, то есть добовляет подключение к репозиторий
на гитхабе.
	git remote add <name: origin> <ссылка_на_репо:ssh>
5. git push - команда для отправки всех ваших изменений на локальном репозитроии в удаленный репозитроий
	git push origin main
----------------------------------------------------------------
	1) git init (только 1 раз)
2) git add .
3) git commit -m ''
	3.1) git branch-M main (1 раз, для переименование ветки)
	*) git remote add origin <ссылка> (1 раз)
4.*) git push origin main
